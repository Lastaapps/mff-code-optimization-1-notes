## Dominance Definition
**DEF:** B DOM C iff every path Entry $\to$ C contains B.

**Properties:**
- Dominators form a tree.
- Complexity: $O(n)$ for tree representation.
- IF B DOM C, THEN B is a predecessor of C in the DFS tree.
