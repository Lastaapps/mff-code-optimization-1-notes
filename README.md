# Code Optimization Course Notes

This repository contains comprehensive lecture notes for the **Code Optimization in Production Compilers** course at MFF UK in **year 2026**.

Blackboard photos were OCR transcribed, then used alongside human-written notes as a source for Gemini 3 Flash to generate notes in `notes/`. Slides were not used (I've found about them to late in the process).
